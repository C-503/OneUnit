# АннотацияВключенНаОС

## ПриСозданииОбъекта

```bsl
// Аннотация для указания операционной системы, на которой должен выполняться тест
//
// Параметры:
//  Значение - Строка - Операционная система, на которой должен выполняться тест,
//   допустимо указывать только один из вариантов: `Windows`, `Linux`, `MacOS`
//  Причина  - Строка - Причина пропуска теста, которая будет указана в отчете о тестировании
//
// Пример:
//  1.
//  // Выполнять только на Windows
//  //
//  &Тест
//  &ВключенНаОС("Windows", "Сам красноглазь")
//   
//  2.
//  // Выполнять только на Linux
//  //
//  &Тест
//  &ВключенНаОС("Linux", "Я и красноглазю")
//   
//  3.
//  // Выполнять только на MacOS
//  //
//  &Тест
//  &ВключенНаОС("MacOS", "Что-то на богатом")
//
Процедура ПриСозданииОбъекта(Значение, Причина = "Выключен в текущей операционной системе") 
```
