# АннотацияИсточникЗначение

## ПриСозданииОбъекта

```bsl
// Значения параметров задаются литералами переданными в текущую аннотацию,
//  значения передаются в параметры метода по порядку, в случае если значение не будет задано
//  то параметр будет иметь значение Неопределено
//
// Параметры:
//  Значение - Строка, Число, Дата, Булево, Неопределено - Значение параметра (Повторяемый)
//
// Пример:
//  1.
//  // Значение первого параметра будет 1 а второго неопределено
//  //
//  &ПараметризованныйТест
//  &ИсточникЗначение(1)
//  Процедура Тест(ПараметрСЕдиницей, ПараметрНеопределенно) Экспорт
//   
//  2.
//  // Значение первого параметра будет "ЯСтрока" а второго 01.01.2020
//  //
//  &ПараметризованныйТест
//  &ИсточникЗначение("ЯСтрока", '20200101')
//  Процедура Тест(ПараметрСоСтрокой, ПараметрСДатой) Экспорт
//
Процедура ПриСозданииОбъекта(Повторяемый = Значение) 
```
