# АннотацияИсточникJSON

## ПриСозданииОбъекта

```bsl
// Значения параметров задаются json массивом объектов,
//  где имя поля объекта соответствует имени параметра а значение поля - значению параметра
//
// Параметры:
//  Значение - Строка - json массив значений параметров
//  Файл     - Строка - Путь к файлу json со значениями параметров, путь разрешается от текущего каталога,
//   в общем случае это каталог из которого запустили выполнение тестов, т.е корень проекта
//
// Пример:
//  1.
//  // Будет зарегистрировано 3 теста с параметрами:
//  // Тест(Параметр1 = 1, Параметр2 = 2)
//  // Тест(Параметр1 = 3, Параметр2 = 4)
//  // Тест(Параметр1 = 5, Параметр2 = 6)
//  //
//  &ПараметризованныйТест
//  &ИсточникJSON(
//      "[
//      |    {""Параметр1"": 1, ""Параметр2"": 2},
//      |    {""Параметр2"": 4, ""Параметр1"": 3},
//      |    {""Параметр1"": 5, ""Параметр2"": 6}
//      |]"
//  )
//  Процедура Тест(Параметр1, Параметр2) Экспорт
//   
//  2.
//  // tests/fixtures/params.json
//   [
//      {"Параметр1": 1, "Параметр2": 2},
//      {"Параметр1": 3, "Параметр2": 4},
//      {"Параметр1": 5, "Параметр2": 6}
//   ]
//   
//  // Будет зарегистрировано 3 теста с параметрами:
//  // Тест(Параметр1 = 1, Параметр2 = 2)
//  // Тест(Параметр1 = 3, Параметр2 = 4)
//  // Тест(Параметр1 = 5, Параметр2 = 6)
//  //
//  &ПараметризованныйТест
//  &ИсточникJSON(Файл = "tests/fixtures/params.json")
//  Процедура ИсточникJSONФайл(Параметр1, Параметр2) Экспорт
//
Процедура ПриСозданииОбъекта(Значение = "", Файл = "") 
```
